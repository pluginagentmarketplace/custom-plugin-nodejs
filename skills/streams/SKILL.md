# Node.js Streams Skill

## Overview
Master Node.js streams for memory-efficient processing of large datasets and real-time data handling.

## Topics Covered

### Stream Types
- Readable streams
- Writable streams
- Duplex streams
- Transform streams
- PassThrough streams

### Stream API
- Pipe and pipeline
- Stream events
- Backpressure handling
- Flowing vs paused mode
- Object mode streams

### Built-in Streams
- fs createReadStream/WriteStream
- HTTP request/response
- process.stdin/stdout
- Compression streams
- Crypto streams

### Transform Patterns
- Custom transform implementation
- Stream composition
- Parser streams
- Encoder/decoder streams
- Filter and map patterns

### Advanced Usage
- Stream.pipeline utility
- stream.finished
- Async iterators
- Web Streams API
- Memory optimization

## Prerequisites
- Node.js fundamentals
- Event emitter basics
- Buffer understanding

## Learning Outcomes
- Process large files efficiently
- Create custom streams
- Handle backpressure
- Build streaming pipelines

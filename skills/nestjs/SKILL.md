- Controllers and routing
- Dependency injection
- Middleware and guards
- Pipes and interceptors

- Prisma with NestJS
- Repository pattern
- Migrations
- Transactions

### Authentication
- Passport integration
- JWT strategy
- Guards and decorators
- Role-based access
- Session management

### API Development
- REST controllers
- Validation pipes
- Exception filters
- OpenAPI documentation
- Versioning

### Microservices
- Transport layers
- Message patterns
- Event patterns
- Hybrid applications
- Queue integration

- Node.js basics
- OOP concepts

## Learning Outcomes
- Build modular applications
- Implement authentication
- Create microservices
- Write testable code

# /explore

Explore all available developer roadmaps, specializations, and learning pathways.

## Usage

```
/explore
/explore backend
/explore react
/explore devops
/explore specializations
```

## Description

Discover and explore the comprehensive catalog of 60+ developer roadmaps and specializations available through the Developer Roadmap Pro plugin.

This command helps you:
- Browse all available career paths and specializations
- Understand prerequisites and skill progressions
- Find the right roadmap for your goals
- See connections between different paths
- Understand job market demand for each path

## What You'll Get

- **60+ Interactive Roadmaps** covering all major tech specializations
- **Career Path Information** including typical salary ranges and job demand
- **Prerequisite Mapping** showing what skills you need to master first
- **Community Insights** from thousands of developers
- **Job Market Data** showing which paths are in demand
- **Difficulty Assessment** for each learning path
